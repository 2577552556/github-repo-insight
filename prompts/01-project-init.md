# Project Initialization

## Prompt Used

Build a full-stack web application that analyzes a public GitHub repository and generates a repository health report.

## Context

The project is a GitHub Repository Health Check application. It should:
- Accept a GitHub repository URL
- Display repository information (stars, forks, issues, etc.)
- Show language distribution with a pie chart
- Calculate and display a health score (0-100)
- Provide AI-powered evaluation with grading

## Tech Stack

- Frontend: Next.js 15, React 19, TypeScript, TailwindCSS, shadcn/ui, Recharts
- Backend: Python 3.12+, FastAPI, Pydantic v2, httpx

## Outcome

This prompt was used to initialize the project structure and establish the foundation for the GitHub Repository Health Check application.
